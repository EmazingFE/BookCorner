# BookCorner 说明：
## 图书来源

Emazing 的每一位小伙伴都是图书角的受益者，所以以自愿为原则，将自己认为质量优秀的书放入书斋库，共同扩充书斋库。

(由于前期没有安全稳定的图书存放点，所以图书放在持有者那里，小伙伴只需提供书名即可。

当然，如果你自己需要阅读，可以随时下架书名；如果经费充足，也可以进行购买。）
